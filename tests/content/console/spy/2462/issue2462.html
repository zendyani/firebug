Issue #2462

This test-case is intended for Issue #2462 - The firebug console still shows the xhr in progress if you abort it (via request.abort()).
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Make sure Show XMLHttpRequests option is on.
  3. Click on the Execute Test button below.
  4. Check the XHR entry. The request must be red (aborted) and the progress spinner stoppped.
