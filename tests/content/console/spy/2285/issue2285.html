Issue #2285

This test-case is intended for Issue #2285 - support for content-type: multipart/x-mixed-replace.
Jan Odvarko, odvarko@gmail.com

You need at least Firefox 3.6 to pass this test.
  1. Open Firebug and enable the Console panel.
  2. Make sure Show XMLHttpRequests option is on.
  3. Firefox -21: Click on the Execute Test button
  4. Firefox 22+: Click on the Execute Test button
  5. Click button below and wait till you see number 4 here: 0
  6. Expand the issue2285.php entry in the Console panel and verify the Response tab. It should show: Part0+Part1+Part2+Part3+
