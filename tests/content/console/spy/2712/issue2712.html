This test-case is intended for Issue #2712 - Gmail has network connections that report 200 Aborted.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Make sure Show XMLHttpRequests option is on.
  3. Click on the Execute Test button below.
  4. Displaed XHR Request must not be marked as Aborted (i.e. not displayed in red).
