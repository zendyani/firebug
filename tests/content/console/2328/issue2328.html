This test-case is intended for Issue #2328 - HTML preview in net-tab but not in console-tab.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Make sure the Show XMLHttpRequests is on.
  3. Click on the Execute Test button below.
  4. There should be one XHR log displayed in the Console panel.
  5. Expand the log and check that HTML tab is available.
