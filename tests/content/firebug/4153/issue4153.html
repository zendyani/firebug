Issue 4153: Re-attaching detached, deactivated Firebug UI doesn't close it

Steps to reproduce

  1. Open Firebug in detached mode on this page.
  2. Deactivate firebug by Selecting "Deactivate Firebug" from the menu or by pressing Shift+F12.
  3. Press Ctrl + F12 to attach Firebug to the main window.

Observed result

  • Firebug opens in browser with blank panel.

Expected result

  • Firebug must be activated before attaching to the main window.
