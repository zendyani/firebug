Copy as cURL: test add --compressed preference

When setting 'extensions.firebug.net.curlAddCompressedArgument' to true the --compressed argument should be added to the generated cURL string
Thomas Andersen, thomas@mr-andersen.no

  1. Navigate to about:config and set 'extensions.firebug.net.curlAddCompressedArgument' to true
  2. Open Firebug, enable the net panel
  3. Click the submit button in the form on this page
  4. Right click on the new entry in the Net panel and execute Copy as cURL
  5. Check the clipboard the cURL command should contain the cURL --compressed argument
