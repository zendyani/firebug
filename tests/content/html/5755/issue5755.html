Issue 5755: Inline editing of cropped attribute values shows the cropped value

  1. Open Firebug
  2. Switch to the HTML panel
  3. Set the displayedAttributeValueLimit pref via about:config to 10

  5. Click the value of the onclick attribute

Expected result

  • The inline editor should open and be filled with whole string of the onclick value.
