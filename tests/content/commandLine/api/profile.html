Command line API: profile, profileEnd

This test-case is intended for command line profile and profileEnd methods.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Verify that profile(); gives a warning about requiring the Script panel.
  3. Enable the Script panel, and reload.
  4. Execute profile() on the command line.
  5. Click the Execute Test buttton.
  6. Execute profileEnd() on the command line.
  7. A table with profile info should be displayed.
