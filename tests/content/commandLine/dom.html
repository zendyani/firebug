Command line: dom

This test-case is intended for console.assert method.
Jan Odvarko, odvarko@gmail.com

  • Open Firebug and enable the Console panel
  • Execute following expressions in the command line and check the results 'expression' -> 'result'
    1. state = true -> true
    2. window -> Window dom.html
    3. document -> Document dom.html
    4. aa -> "three"
    5. cc -> null
    6. dd -> 2
    7. dd = 999 -> 999 and consequently dd -> 999
    8. set_d_val -> set_d_val(td)
    9. set_d_val(998) -> 998
    10. blah -> ReferenceError: blah is not defined { message="blah is not defined", more...}
    11. var blah = 'oink' -> >>> var blah = 'oink' and consequently blah -> "oink"
    12. document.getElementsByTagName('h1')[0] -> "<h1>"
Except of #11 the >>> ... logs are not mentioned, but also expected.
