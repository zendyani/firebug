Issue #4234

This test-case is intended for Issue #4234 - Object with array property showing as blank.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Type following in to the command line: var a = {arr: [1,2,3]}; a;
  3. Press Enter
  4. The result should be: 
    >>> var a = {arr: [1,2,3]}; a;
Object { arr=[3] }
